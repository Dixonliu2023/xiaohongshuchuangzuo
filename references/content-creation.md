## 二、选题方法论

### 优质选题4步法（谢无敌）

**Step 1：找痛点**

| 来源 | 方法 |
|------|------|
| 评论区 | 用户问什么最多 |
| 私信 | 用户最关心什么 |
| 搜索框 | 用户在搜什么 |
| 竞品 | 同行爆款是什么 |

**Step 2：找需求**

| 类型 | 示例 |
|------|------|
| 求知型 | 怎么选、怎么做 |
| 求快型 | 3分钟学会 |
| 求便宜型 | 性价比推荐 |
| 求认同型 | 踩坑分享 |

**Step 3：找热点**

| 渠道 | 方法 |
|------|------|
| 热搜榜 | 找相关话题 |
| 节日 | 提前规划内容 |
| 电影综艺 | 找可蹭热点 |
| 平台活动 | 参加官方活动 |

**Step 4：做测试**

- 先发测试选题
- 根据数据优化
- 找到爆款规律

## 三、标题方法论

### 6大标题法则（姜姜）

#### 1. 数字法则

```
用具体数字增加可信度

案例：
- 3天瘦5斤
- 5个方法搞定
- 100块搞定全身穿搭
```

#### 2. 痛点法则

```
戳中用户痛点

案例：
- 毛孔粗大怎么办
- 预算不够怎么玩
- 路痴怎么旅游
```

#### 3. 身份法则

```
标签人群定位

案例：
- 打工人的一天
- 学生党必看
- 宝妈必收
```

#### 4. 结果法则

```
承诺具体结果

案例：
- 学会这个月瘦10斤
- 看完就会做
- 用了皮肤变好了
```

#### 5. 对比法则

```
前后对比/对比差异

案例：
- 以前vs现在
- 买对vs买错
- 丑小鸭变白天鹅
```

#### 6. 悬念法则

```
引发好奇

案例：
- 没想到
- 竟然
- 真相是
```

---

### 标题公式2.0（整合版）

```
标题 = [价值感知/身份转变] + [核心关键词] + [价值时间线]

案例对比：

原版：五台山旅游攻略
2.0版：徒步小白也能完成的2天朝台攻略

原版：朝台要多少钱
2.0版：五台山大朝台2日游，人均845元全包
```

## 四、封面方法论

### 高点击率封面4要素

| 要素 | 说明 | 示例 |
|------|------|------|
| 大字标题 | 清晰可见 | 攻略、合集 |
| 人物出镜 | 增加亲和力 | 网红/素人 |
| 场景背景 | 增加真实感 | 实地拍摄 |
| 色彩对比 | 吸引注意 | 明亮/高对比 |

---

### 4种爆款封面类型

#### 1. 合集类

```
特点：信息量大，显得专业
结构：背景图 + 主标题 + 副标题 + 元素

案例：
- 背景：虚化场景图
- 主标题：朝台攻略
- 副标题：2天45公里
```

#### 2. 大字报类

```
特点：简洁有力，戳中痛点
结构：大留白 + 大字标题 + 表情包

案例：
- 背景：大留白
- 表情包：惊讶/无语
- 大字：小白也能完成
```

#### 3. 拼图类

```
特点：展示丰富内容
结构：多图拼接 + 标注文字

案例：
- 6宫格拼图
- 标注：早/午/晚
```

#### 4. 人物类

```
特点：有亲和力，增加信任
结构：人物照片 + 主标题 + 效果展示

案例：
- 穿搭分享
- 前后对比
```

## 五、正文方法论

### 万能正文结构

```
【开头】价值前置
- 痛点切入/场景引入
- 给出价值承诺

【中间】干货内容
- 1234要点罗列
- 每点有说明
- 可适当植入产品

【结尾】引导互动
- 总结要点
- 引导收藏
- 引导评论
```

---

### 开头3秒法则

**目的：让用户知道"看完能获得什么"**

**开头方式：**

1. **价值前置**
   ```
   3分钟就能学会的XXX，看完就能用
   ```

2. **痛点切入**
   ```
   是不是觉得朝台很难？我来告诉你
   ```

3. **身份认同**
   ```
   作为过来人，这些经验必须告诉你
   ```

---

### 正文写作技巧

**1. 善用数字**
- "3个方法" > "一些方法"
- "45公里" > "很长"

**2. 善用emoji**
- 分隔段落
- 突出重点
- 增加趣味

**3. 段落简短**
- 每段不超过3行
- 手机阅读友好

**4. 干货充足**
- 有实用价值
- 有具体步骤
- 有数据支撑

## 六、违禁词检测

### 6大类违禁词

| 类别 | 违禁词 | 替代表达 |
|------|--------|----------|
| 极限词 | 最、第一 | 非常、很 |
| 医疗词 | 治疗、治愈 | 改善、调理 |
| 引流词 | 微信、加我 | 戳我、评论区 |
| 营销词 | 特价、秒杀 | 优惠、活动价 |
| 虚假词 | 无效退款 | 试用不满意退 |
| 违规词 | 代购、A货 | 海外直邮 |

### 笔记体检状态灯

#### 🟢 绿色（可发布）
- 封面一眼看到价值
- 标题有明确时间线
- 内容有具体可操作步骤

#### 🟡 黄色（建议优化）
- 内容有用，但价值需要看完才明白

#### 🔴 红色（不建议发布）
- 纯记录生活，无读者获得感

## 八、内容配比规划（新增）

基于工业化运营策略，极度垂直的流量漏斗。

### 1. 70-20-10内容配比

**核心理念：**
| 占比 | 内容类型 | 作用 | 目的 |
|------|----------|------|------|
| 70% | 干货攻略 | 吸引搜索流量 | 涨粉基石 |
| 20% | 案例展示 | 建立信任 | 转化基础 |
| 10% | 热点话题 | 获取曝光 | 流量补充 |

### 2. 各类型内容详解

**70%干货内容：**
- 路线攻略、教程指南
- 清单合集、避坑指南
- 费用明细、装备推荐
- 特点：提供实用价值，吸引精准用户

**20%案例内容：**
- 领队日常、服务过程
- 客户反馈、成功案例
- 前后对比、效果展示
- 特点：建立信任，为变现做铺垫

**10%热点内容：**
- 节假日出行、季节景色
- 行业新闻、平台活动
- 话题蹭流、时效内容
- 特点：获取泛流量，扩大曝光

### 3. 五台山赛道配比应用

| 占比 | 内容 | 具体选题 |
|------|------|----------|
| 70% | 干货 | 路线攻略、祈福指南、装备清单、费用明细 |
| 20% | 案例 | 领队日常、客户好评、祈福案例、带团记录 |
| 10% | 热点 | 五一出行、国庆攻略、冬季景色、寺庙新闻 |

### 4. 周内容规划模板

```
周一：干货 - 路线攻略
周二：干货 - 祈福指南
周三：案例 - 领队日常
周四：干货 - 装备清单
周五：热点 - 节假日
周六：案例 - 客户反馈
周日：干货 - 费用攻略
```

### 5. 内容配比执行要点

**注意事项：**
- 不要为了配比而配比
- 优先保证内容质量
- 根据数据反馈调整
- 变现期可适当增加案例比例

**优化方向：**
- 干货数据好 → 增加干货比例
- 案例转化高 → 增加案例内容
- 热点流量大 → 蹭热点增加曝光
