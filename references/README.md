## 核心公式速查

### 价值实现框架
```
价值四维 = 价值清晰度 + 价值时间线 + 价值感知 + 价值发现
CES 2.0 = 点赞×1 + 收藏×1 + 评论×4 + 转发×4 + 关注×8
核心心法：生意成功的关键在于用户能否清晰表达获得的价值
```

### IP万能公式
```
我要给 [什么样的人群] 提供 [什么样的内容/服务]，通过 [什么方式] 变现
```

### CES评分公式
```
CES评分 = 点赞×1 + 收藏×1 + 评论×4 + 转发×4 + 关注×8
```

### 标题公式
```
标题 = 修饰词 + 核心关键词
```

### 违禁词6大类
1. 极限词（最、第一、顶级）
2. 医疗敏感词（治疗、治愈、秘方）
3. 引流私域词（微信、加我、私信）
4. 营销敏感词（特价、秒杀、最低）
5. 虚假宣传词（无效退款、保证有效）
6. 违规交易词（代购、A货、高仿）

### 价值四维度（新增）
1. 价值清晰度：读者能否一眼看出获得什么
2. 价值时间线：即时满足 vs 延迟价值
3. 价值感知：封面/开头是否传递价值
4. 价值发现：搜索流量 vs 推荐流量

### 笔记体检状态灯（新增）
- 🟢 绿色：封面一眼看到价值，可发布
- 🟡 黄色：价值感知不够，建议优化
- 🔴 红色：无读者获得感，重新调整

## 常见问题

### Q：如何快速生成标题？

A：
1. 确定核心关键词
2. 套用 title-formula.md 中的8个模板
3. 生成5-10个选项
4. 选择最佳标题

### Q：笔记写什么类型？

A：根据 content-templates.md 中的6种笔记类型选择：
1. 干货合集 → 信息整理
2. 资料引流 → 送资料
3. 产品种草 → 推荐产品
4. 案例展示 → 展示成果
5. 体验分享 → 真实体验
6. 故事名片 → 个人故事

### Q：如何检测违禁词？

A：
1. 将笔记内容输入违禁词检测功能
2. 系统自动扫描6大类违禁词
3. 标注违禁词位置
4. 替换为替代表达
5. 二次检测确认

### Q：如何找选题？

A：使用 topic-mining.md 中的4种方法：
1. 搜索关键词法
2. 对标博主法
3. 用户反馈法
4. 官方渠道法

### Q：如何拆解爆款？

A：使用 analytics.md 中的7维度拆解：
1. 封面分析
2. 标题分析
3. 选题分析
4. 内容分析
5. 文本框分析
6. 数据分析
7. 价值分析（新增）

### Q：如何评估选题价值？（新增）

A：使用 value-engine.md 中的价值四维评估：
1. 价值清晰度：选题是否明确"获得感"
2. 价值时间线：即时满足还是延迟价值
3. 价值感知：封面是否一眼看到价值
4. 价值发现：搜索流量还是推荐流量

### Q：笔记价值体检怎么做？

A：使用 forbidden-words.md 中的状态灯：
- 🟢 绿色：可发布
- 🟡 黄色：优化后发布
- 🔴 红色：重新调整

### Q：如何升级标题？

A：使用 content-templates.md 中的标题公式2.0：
- 原公式：标题 = 修饰词 + 核心关键词
- 2.0版：标题 = 价值感知 + 核心关键词 + 价值时间线
