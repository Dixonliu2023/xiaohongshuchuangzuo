# 图像提示词参考

这个文档服务于主 Skill 的封面生成、海报生成和人物参考迁移场景。

它只解决一类问题：
- 已经想清楚这张图要服务什么内容
- 现在要把画面提示词写得更准

它不替代下面这些判断：
- 这篇内容写给谁
- 用户为什么会点
- 封面冲突是什么
- 标题和正文承诺是否一致

---

## 一、什么时候用

适合在这些场景调用：
- 要做小红书 3:4 封面图
- 要参考某个人物形象生成封面
- 要把“旅行产品”变成更有主体感的首图
- 要生成纪实感、海报感、路线感更明确的视觉图

不适合在这些场景代替主判断：
- 还没想清楚用户是谁
- 还没想清楚标题和封面大字
- 还没想清楚这篇是收客、信任还是种草

---

## 二、默认写法顺序

写图像 prompt 时，按这个顺序表达最稳：

1. 用途
2. 画幅比例
3. 主体是谁
4. 场景是什么
5. 镜头距离和构图
6. 风格气质
7. 光线和色调
8. 不要什么
9. 文字排版内容

简化模板：

```text
[平台用途] + [比例]
[主体身份/人物特征]
[场景]
[镜头距离/站位/构图]
[风格]
[光线/色调]
[不要的元素]
[封面文字]
```

## 三、小红书封面专用模板

### 1. 通用封面模板

```text
小红书封面图，3:4 竖版。
画面主体是[人物/路线/产品场景]，服务于[选题主题]。
整体风格真实、克制、可信，不要廉价广告感，不要过度网红感。
主体位于[左侧/中间/右侧]，画面保留清晰文字排版区域。
场景为[山野/城市/茶馆/车内/步道/民宿/夜景]。
镜头为[半身近景/中景/广角全身]。
色调为[冷灰蓝/暖黄/自然肤色/纪录片感]。
画面强调[决策感/松弛感/可信感/路线判断感]。
封面上直接排版中文：
主标题“……”
副标题“……”
小字“……”
字体清晰，信息层级明确。
```

### 2. 旅行收客封面模板

```text
小红书旅行封面，3:4 竖版。
不是纯风景海报，而是带判断感的路线封面。
画面主体是[人物/路线节点/地图感场景]。
让人一眼看出这是在回答“我适不适合”“我该怎么选”。
整体风格真实、可信、纪实，不要旅行社廉价宣传单风格。
保留大字标题区域，适合小红书首图点击。
```

## 四、人物参考迁移怎么写

如果用户给了一张参考人物图，不要只写“照着这个人生成”。
要拆成 4 层：

1. 外观特征
2. 气质状态
3. 使用场景
4. 保留边界

### 参考模板

```text
参考用户提供的人物形象：
中国男性，短发，戴细框眼镜，脸型和五官气质接近参考图。
人物状态成熟、稳重、可信，不要明星感，不要过度精修。
保留普通人真实质感，略带疲惫但坚定。
把人物放进[山野徒步/城市街头/茶馆/车厢/民宿]场景中。
不要变成过度健身、过度年轻化或网红摄影风。
```

### 适合你的人设表达

你现在的 IP 不是“硬核户外博主”，也不是“文艺旅行摄影师”。
更适合这些词：
- 成熟稳重
- 可信
- 带判断感
- 理解高压人群
- 有状态切换经验
- 不喊口号
- 不装松弛

## 五、旅行内容的风格词库

### 1. 大朝台/徒步类

适合：
- 纪录片感
- 冷灰蓝山景
- 高海拔云雾
- 风大、冷冽、真实
- 山脊步道
- 人在路上
- 微疲惫但坚定
- 克制，不神化

不适合：
- 廉价佛系素材拼贴
- 过重宗教视觉堆砌
- 假装史诗大片
- 纯景区海报感

### 2. 成都松弛/状态切换类

适合：
- 暖色城市傍晚
- 茶馆、人间烟火、后备箱酒馆
- 放松但不懒散
- 有生活质感
- 微电影感
- 真实城市切换感

不适合：
- 假精致摆拍
- 过度文青滤镜
- 空洞松弛感

### 3. 定制游/包车类

适合：
- 可信的路线感
- 清楚的节点信息
- 有人带路的安全感
- 不折腾
- 节奏更稳

不适合：
- 全是豪车、奢华、假高端
- 过度销售海报

## 六、镜头和构图表达法

### 镜头距离

- 半身近景：适合做人设、判断、可信感
- 中景：适合人物和环境一起交代
- 全身广角：适合路线、徒步、城市空间感

### 构图表达

- 人物在左，右侧留文案：适合判断型封面
- 人物居中，顶部大字：适合故事/观点型封面
- 下方路线，上方天空留白：适合徒步/远行感
- 前景人物，后景路线：适合“我来带你判断”的封面

### 文字预留表达

直接写清楚：
- 右上角留白用于主标题
- 左侧中部保留 3 行大字区域
- 下方保留副标题区

不要只说“适合排版”，要说清楚留白在哪。

---

## 七、常用风格限制词

为了避免图像生成偏掉，可以加入这些限制：

- 不要廉价旅游广告感
- 不要网红摆拍感
- 不要过度磨皮
- 不要塑料感皮肤
- 不要假笑
- 不要过饱和
- 不要太多无关元素
- 不要卡通感
- 不要廉价字体
- 不要把画面做成景区宣传单

---

## 八、封面文字怎么放进提示词

如果要直接在图里出字，建议明确 3 层：

```text
封面上直接排版中文：
主标题“……”
副标题“……”
右下角小字“……”
字体有力量但不过度商业化，信息层级清晰，排版干净。
```

如果担心模型出字不稳，可以只写：
- 预留主标题区
- 预留副标题区
- 预留角标区

后续再手动加字。

## 九、适合你业务的 4 个即用模板

### 1. 大朝台判断型

```text
小红书封面，3:4 竖版。五台山大朝台主题，不是景区宣传海报，而是第一次朝台的路线判断封面。参考用户提供的人物形象：中国男性，短发，戴眼镜，成熟稳重，可信，略带疲惫但坚定。人物位于画面左侧中景，背景是冷灰蓝色山脊步道与云雾。整体真实、克制、纪录片感，不要廉价旅游广告感。右侧预留大字排版区域。封面上直接排版中文：主标题“第一次走五台山大朝台” 副标题“先别急着选78km” 小字“普通人先看60km还是78km”。
```

### 2. 成都状态切换型

```text
小红书封面，3:4 竖版。成都城市松弛主题，服务于“高压打工人状态切换”。画面是傍晚暖色街头或茶馆场景，人物处于放松但不松垮的状态，真实、有生活感，不要空洞文艺。保留标题区，适合小红书首图。整体像真实城市纪录片封面，不要网红摆拍。
```

### 3. 包车定制信任型

```text
小红书封面，3:4 竖版。私人定制游/包车游主题，不是销售海报，而是可信的路线判断封面。画面同时交代人物与路上场景，强调省心、稳、有人带路。整体真实、干净、可信，不要奢华假高端，不要过度商业化。
```

### 4. 路线对比型

```text
小红书首图，3:4 竖版。路线对比主题，适合“60km vs 78km”“2天 vs 4天”这种判断内容。画面要有清晰对比感，可以是两条路线、两种状态或两侧不同难度感受。整体强调决策冲突，不强调纯风景。适合放大字标题，信息层级强。
```

---

## 十、使用提醒

- 图像 prompt 只负责把画面说清楚，不负责替你完成内容判断
- 先有人群、冲突、标题，再来写图
- 对旅行收客内容来说，真实可信通常比“炸裂好看”更重要
- 不要为了画面感，牺牲对象感和转化感
